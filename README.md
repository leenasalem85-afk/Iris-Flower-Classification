# Iris Flower Classification

## Overview
A Python machine learning project that classifies Iris flowers into three species (Setosa, Versicolor, Virginica)
based on four measurements: sepal length/width and petal length/width.

## Features
- Data exploration and visualization
- Decision Tree model training and evaluation
- OOP-based structure (training, prediction)
- Simple Tkinter GUI for user input and instant prediction

## Tech Stack
- Python
- Pandas, Matplotlib
- scikit-learn
- Tkinter
